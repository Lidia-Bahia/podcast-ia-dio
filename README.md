# 🎙 Arte Sintética  
### Humanos criando, máquinas aprendendo — e o limite entre os dois.

Bem-vinda(o) ao repositório oficial do podcast **Arte Sintética**, trazendo reflexões sobre criatividade em tempos de inteligência artificial. 
Aqui, conversamos sobre arte, música, vídeo, autoria, cultura digital e do paradoxo de **criticar IA enquanto se usa IA**.

---

## 🛠 Tecnologias & Ferramentas comuns

O episódio foi criado utilizando ferramentas como:

- **ChatGPT** (roteiro)  
- **ElevenLabs** (narração)  
- **Leonardo AI** (arte visual)  
- **Capcut** (edição)
